# NextStep – Smart Wedding Planning & Service Platform


## Problem

Users struggle to plan weddings due to:

* Scattered vendor information
* No structured planning workflow
* Lack of personalized recommendations

## Solution

NextStep provides:

* End-to-end wedding planning system
* Vendor discovery and booking
* Personalized recommendations
* Real-time chat and notifications

---

## Core Features

### 1. Planning System

* Step-by-step wedding planning workflow
* Checklist-based progress tracking
* Dynamic planner updates

### 2. Booking System

* Vendor/service booking
* Booking lifecycle management (create → confirm → complete)
* Payment integration handling

### 3. Recommendation Engine

* Suggests vendors/services based on:

  * User preferences
  * Budget
  * Event requirements

### 4. Chat System

* Real-time communication between users and vendors

### 5. Notification System

* Event-based notifications for:

  * Bookings
  * Messages
  * Updates

---

## System Architecture

### Backend

* Node.js + Express
* Modular structure:

  * Controllers (business logic)
  * Models (data structure)
  * Middleware (auth & access control)
  * Utils (core logic)

### Frontend

* React-based architecture
* Custom hooks for logic handling
* Service layer for API communication

## How It Works (Flow)

1. User signs up and creates a wedding plan
2. System generates checklist & planner
3. User receives recommendations
4. User books vendors/services
5. Chat + notifications manage communication
6. System tracks progress until completion
